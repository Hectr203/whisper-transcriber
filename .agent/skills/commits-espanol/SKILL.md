# Generación de Commits en Español

## Objetivo
Garantizar que todos los mensajes, descripciones y comentarios relacionados con repositorios de Git y commits sean generados estrictamente en español.

## Reglas y Restricciones
- Al generar un mensaje de commit, el contenido principal debe estar en español.
- Si se usa la convención de "Conventional Commits" (ej. `feat:`, `fix:`), el prefijo puede mantenerse en inglés, pero el texto descriptivo debe ir en español. Ejemplo: `feat: agrega nueva funcionalidad de login`.
- Las explicaciones, PR descripciones o cualquier documentación relacionada con un commit también debe escribirse en español.
