# Referencia: vtptransportes

## Uso permitido
El proyecto `vtptransportes` puede usarse como ejemplo de proyecto existente con frontend, backend, configuraciones, dependencias y control de versiones propio.

## Limite
Esta agencia no esta disenada exclusivamente para `vtptransportes`. Ninguna decision debe generalizarse desde ese proyecto hacia todos los demas sin verificar primero el repositorio real sobre el que se trabaja.

## Que observar como referencia
- Separacion entre frontend y backend.
- Existencia de dependencias ya seleccionadas.
- Configuraciones y archivos de entorno.
- Estructura de carpetas previa.
- Posible historia de cambios en Git.

## Regla
Usar `vtptransportes` solo para entender el tipo de contexto que puede tener un proyecto avanzado. No copiar su stack, estructura ni reglas si el proyecto objetivo es diferente.
