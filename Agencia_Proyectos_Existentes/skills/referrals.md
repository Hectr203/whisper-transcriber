# Skill: Referrals

## Proposito
Apoyar procesos de referidos, recomendaciones, candidatos o comunicaciones relacionadas cuando el dominio del proyecto lo requiera.

## Correspondencia en `agent-skills`
No existe una skill equivalente dedicada a referidos o reclutamiento.

## Referencia mas cercana
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/spec-driven-development/SKILL.md`
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/documentation-and-adrs/SKILL.md`

## Rol en el sistema
Esta skill permanece como una especialidad de dominio. El nucleo solo aporta la forma de estructurar reglas, actores y criterios, no un flujo de referidos especifico.

## Cuando usar
- Sistemas de reclutamiento.
- Programas de referidos.
- Seguimiento de candidatos o contactos.

## Procedimiento
1. Identificar flujo de referido.
2. Definir actores y estados.
3. Proteger datos personales.
4. Documentar criterios de avance.

## Salidas
- Flujo funcional.
- Reglas de negocio.
- Riesgos de privacidad.
