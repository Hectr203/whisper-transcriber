# Skill: Creador de Habilidades

## Proposito
Crear nuevas skills cuando exista una necesidad recurrente.

## Correspondencia en `agent-skills`
No existe una skill equivalente dedicada a crear skills.

## Referencia mas cercana
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/spec-driven-development/SKILL.md`

## Rol en el sistema
Esta skill sigue siendo una utilidad de authoring del proyecto. La referencia del nucleo mas cercana sirve para estructurar objetivo, limites y salida, pero no reemplaza la tarea de disenar una skill nueva.

## Cuando usar
- Un flujo se repite varias veces.
- Hay reglas especializadas de un dominio.
- Se necesita estandarizar una practica.

## Procedimiento
1. Definir objetivo.
2. Establecer cuando usar y cuando no usar.
3. Documentar entradas, pasos, salidas y limites.
4. Agregar combinaciones recomendadas.

## Salidas
- Nueva skill documentada.
