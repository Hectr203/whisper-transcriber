# Skill: ui-ux-pro-max

## Proposito
Mejorar experiencia de usuario, claridad visual, accesibilidad y eficiencia operativa.

## Base principal
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/frontend-ui-engineering/SKILL.md`
- Implementacion local complementaria: `Agencia_Proyectos_Existentes/skills/ui-ux-pro-max/SKILL.md`

## Rol en el sistema
Esta skill conecta el criterio de UI del nucleo con el banco de estilos, datos y scripts locales del proyecto. Su responsabilidad es mantener el diseno original cuando exista y mejorar solo lo necesario.

## Cuando usar
- Interfaces.
- Formularios.
- Dashboards.
- Flujos de navegacion.
- Estados de carga, error o vacio.

## Procedimiento
1. Identificar usuario objetivo y tarea principal.
2. Reutilizar sistema visual existente.
3. Priorizar claridad, legibilidad y accesibilidad.
4. Verificar mobile y desktop si aplica.

## Salidas
- UI coherente.
- Estados completos.
- Recomendaciones de accesibilidad.
