# Skill: Lectura y Comprension de Arquitectura Existente

## Proposito
Entender capas, modulos, dependencias y contratos antes de cambiar codigo.

## Correspondencia en `agent-skills`
No existe una skill exacta para lectura de arquitectura existente.

## Rol en el sistema
Esta skill es la version local para mapear un proyecto desconocido o ya existente. Su foco es descubrir convenciones reales antes de modificar nada.

## Cuando usar
- Cambios backend, frontend, base de datos o integraciones.
- Refactorizaciones.
- Proyectos desconocidos.

## Procedimiento
1. Identificar puntos de entrada.
2. Mapear flujo de datos.
3. Revisar dependencias entre capas.
4. Detectar patrones repetidos.
5. Registrar reglas verificadas.

## Salidas
- Mapa de arquitectura.
- Convenciones detectadas.
- Riesgos de impacto.
