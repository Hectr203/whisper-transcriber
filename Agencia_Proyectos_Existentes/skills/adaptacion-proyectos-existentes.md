# Skill: Adaptacion a Proyectos Existentes

## Proposito
Trabajar sobre codigo ya construido sin imponer una estructura externa.

## Base principal
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/context-engineering/SKILL.md`
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/code-review-and-quality/SKILL.md`

## Rol en el sistema
Esta skill es la capa local para entrar a un repo existente, leer sus reglas reales y adaptar el cambio al sistema ya construido.

## Cuando usar
Siempre que la tarea se realice dentro de un repositorio existente.

## Procedimiento
1. Leer documentacion y configuraciones.
2. Identificar tecnologias y convenciones.
3. Buscar ejemplos equivalentes.
4. Definir si conviene conservar, adaptar, extender o refactorizar.
5. Aplicar cambios incrementales.

## Salidas
- Estrategia de modificacion.
- Riesgos.
- Validaciones.

## Limites
No usar para justificar reestructuraciones amplias sin evidencia.
