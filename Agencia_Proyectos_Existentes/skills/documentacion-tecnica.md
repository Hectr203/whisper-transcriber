# Skill: Documentacion Tecnica

## Proposito
Registrar decisiones, cambios y conocimiento reutilizable.

## Base principal
- `Agencia_Proyectos_Existentes/skills/agent-skills/skills/documentation-and-adrs/SKILL.md`

## Rol en el sistema
Esta skill es la version local para documentar el por que de los cambios, con especial atencion a decisiones tecnicas, ADRs y bitacoras reutilizables.

## Cuando usar
- ADRs.
- Cambios de arquitectura.
- Cierre de tareas.
- Guias de uso.

## Procedimiento
1. Documentar contexto.
2. Explicar decision y alternativas.
3. Registrar consecuencias.
4. Separar hechos de pendientes.

## Salidas
- Informe.
- ADR.
- Guia o bitacora.
